Enviar parametros del request http GET - URL

En thymeleaf para las rutas se indica con "@" en lugar de "#" y los parametros entre ()

  • /params/string?texto=Valor de parametro
  • /params/string?texto=Vamos Irving!
  • /params/mix-params?saludo=Hola Irving&num=26
  • /params/mix-params-request?saludo=Hola Irving&num=26
